# T-Rex_JavaFX
## Description:
Ucode project: develop minigame like Google T-REX game.

## Usage:
1. git clone https://github.com/nnocturnnn/T-Rex_JavaFX
2. cd T-Rex_JavaFX
3. ./gradlew build
4. ./gradlew run
